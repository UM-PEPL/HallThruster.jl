Verification

Tests can be found in the test folder, and are split in unit_tests and order_verification tests. The julia Test environment is used. Order verification studies verify the correct implemenation of the numerics by comparing the theoretical to the actual order of accuracy of the spatial discretization. For more details on the discretization, see Fluxes and Numerics.
